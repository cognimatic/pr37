Delimiter-based widget for multivalue text fields

This is a tweaked version of an unmaintained module https://www.drupal.org/project/multi_text - not maintained for 6 years so needed tweaks to work with D9

If the module is ever updated, remove/delete this 'custom' module and install the managed version using Composer. Probably don't need to uninstall before changing over, but worth backing everything up just in case
